# wardrobeProyecto: Sistema Inteligente de Outfits — Ángel (Roscas84)
Qué es:

Una app web personal para generar outfits combinados por colorimetría, basada en el inventario real de ropa fotográfica.
Dónde vive:

GitHub: github.com/Roscas84/wardrobe
URL pública desde iPhone: https://roscas84.github.io/wardrobe

Qué tiene hasta ahora:

App funcional publicada en GitHub Pages
Inventario con foto real, nombre, tipo, color HEX, marca y formalidad
Constructor de outfits en 3 pasos: prenda ancla → capas arriba/abajo → accesorios → resultados
Motor cromático basado en rueda de color: monocromática, análoga, complementaria, complementaria dividida, triádica y tetrádica
Fotos reales de las prendas en cada outfit generado
Almacenamiento local en el teléfono (localStorage)

Inventario cargado:
